构造函数与析构函数

  1. 一个类中析构函数只能有一个，而构造函数可以有很多个
  2. 析构函数没有函数参数、没有返回值、没有返回类型
  3. 由于没有函数参数，所以不能被重载
  4. 构造函数在类的构建时调用，析构函数在类的声明周期结束时调用
